How Does It Work?

Audio goes in. Video goes out.
(Audio out is optional.)

The software is written in a creative coding language called Processing.

The hardware is based on a powerful microcomputer called Raspberry Pi.

How Do I Operate It?

The device turns on and off with a power switch.
It has two knobs that adjust the audio input and output volumes.

You can play, pause, and switch between programs using a USB keyboard, a MIDI device, an X-Box control pad, or just about anything else.

Can I Use My Own Music Videos?

VIZBOX can play any standard video file. If you want to play along with the sick vids you've already got, we'll put 'em in a compact, easy-to-use package.

Alternately, we can chop, loop, and manipulate your existing visuals, so they're more flexible and responsive in a live situation.
